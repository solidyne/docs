    | ------------------------- | ---------: | -------------------------------------------- |

    | Linha estéreo balanceada  | 4         | Nível + 4 dBu                               |

    | Tipo          | Quantidade           | Detalhe                                               |

    | USB estéreo   | 4                  | Quatro dispositivos de reprodução em um computador     |
    | UNIDEX        | 16 mono (8 estéreo) | Até 16 microfones adicionais ou 8 dispositivos TELCO  |

    | Processamento | Detalhe                        |
    | ------------ | ------------------------------ |
    | DSP          | Processos DSP em cada entrada  |

    | Função                  | Especificação                                                        |
    | ----------------------- | ------------------------------------------------------------------- |
    | Luzes "No Ar"          | 2 saídas (Estúdios A e B), 12 VCC, 250 mA                          |
    | GPI                     | 4 entradas de uso geral (ativa alta 3.3 ~ 15 V)                     |

  <Accordion title="Arquitetura e processamento de áudio">
    | Parâmetro             | Valor                                                                                                           |
    | --------------------- | --------------------------------------------------------------------------------------------------------------- |
    | Buses de mixagem      | 4 buses estéreo (PGM, REC, AUX com atribuição direta da superfície; VBUS atribuível pela tela)                 |
    | PFL                   | 1 bus PFL (CUE)                                                                                                 |
    | Resolução / SR        | 24 bits @ 48 kHz                                                                                                |
    | Processos DSP         | Fixos por entrada e por tipo: EQ paramétrico (3 ou 4 bandas), gate de ruído, compressor/limitador, de-esser    |
    | Mix-minus / talkback  | Envios automáticos (TELCO)                                                                                      |
    | Latência              | 1.2 ms (linha analógica → saída analógica) / 1.4 ms (MIC com EQ + dinâmica + de-esser)                         |
  </Accordion>
