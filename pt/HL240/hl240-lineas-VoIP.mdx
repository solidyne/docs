As linhas telefônicas digitais sobre IP (voz sobre IP ou VoIP) podem ser conectadas às entradas analógicas RJ11 usando adaptadores VoIP para par telefônico. Estes dispositivos também são conhecidos como **Adaptador de Telefone Analógico** (ATA)

Outra alternativa é utilizar um telefone VoIP com suporte Bluetooth para viva-voz. Neste caso, usaremos o modo "Viva-voz" para vincular o telefone a um canal Bluetooth do Telco-Mixer HL240-VQ.

![Hl240 Diagrama Lineas Pn](/images/hl240/hl240-diagrama-lineas.png)

Para informações específicas sobre estes dispositivos, consulte a documentação do fabricante.
