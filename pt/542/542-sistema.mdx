## Ajustes gerais do sistema

![542 SYSTEM Config](/images/542/542-system_general.png)

### CONFIG GPIO

- **GPI VOICE CHANGE PRESET**:\
  Quando a GPI é ativada, alterna o preset de processamento.
  - O sinal de disparo é gerado no Estúdio ao habilitar os microfones.
  - Pode chegar ao processador por conexão **hardware** ou via **LAN** (modelos com opção /AoIP).
  - O campo à direita define o preset a ser aplicado (padrão: **VOICE-LOUD**).
- **GPO TRIGGER**:\
  Lista de eventos que podem ativar a saída GPO:
  1. SILÊNCIO DE ÁUDIO
  2. ENTRADA FAIL BACKUP
  3. SOBREMODULAÇÃO
  4. BAIXA POTÊNCIA RF
  5. FORçar FM MONO

<Info>
  Ver detalhes de [conexão das GPIO](/pt/unidex/instalacion-gpio)
</Info>

### SEGURANÇA

Permite habilitar senha desmarcando a opção **Disable Password Protection** (padrão: ativado).

- Senha padrão: `1234`.
- Para modificar, digite no campo **PASSWORD** e pressione **APPLY**.
- Regras: até **8 caracteres**, apenas letras **MAIÚSCULAS** e números.
- **FRONT PANEL CHANGES (LOCK/UNLOCK)**:\
  Bloqueia ou habilita o acesso pelo painel frontal.\
  Se bloqueado, é necessária a senha para modificar pelo painel.

---

### REDE

Configuração de rede.

- De fábrica: modo **DHCP habilitado**.
- Pode ser atribuído um **IP estático** desativando o DHCP.
- O endereço IP é informado no painel frontal do equipamento ou via LAN com a ferramenta **Solidyne Multi-Discovery**.

<Info>
  Para mais detalhes, veja [Conexão e acesso Ethernet](/pt/542/542-instalacion-web).
</Info>

### Interface do usuário

- **LOUDNESS EBU REFERENCE**: Nível de referência de sonoridade no monitoramento geral (padrão: 0 dBfs).
- **AUTO-OPEN LEFT MENU**: Expande automaticamente o menu ao aproximar o mouse.
- **ENABLE TOOLTIPS**: Ativa mensagens de ajuda pop-up sobre controles e indicadores.

### RELATÓRIO TÉCNICO

Gera um relatório com parâmetros de funcionamento e configuração do equipamento.

- Pressionando "DOWNLOAD", o relatório é salvo na pasta de downloads do navegador.
- Pode ser enviado ao **suporte técnico** para diagnóstico.
