### Uso

- À medida que o **programa de ar** reproduz títulos, ele notifica o módulo RDS.
- O módulo acessa o banco de dados para obter as informações necessárias.

Funções disponíveis:

- **Mensagens em tempo real**: escreva o texto no campo _Flash Msg_ e pressione **[Flash]**.
- **Anúncio de tráfego**:
  - Ative a caixa _Send as traffic announce_.
  - Alguns receptores RDS exibem simultaneamente a informação principal e o anúncio de tráfego; outros alternam entre ambas.
- **Configuração diferenciada**: é possível definir informações diferentes para música, blocos comerciais ou outros blocos.
- **Exportar info para**:
  - Despeja a informação gerada pelo RDS em um arquivo de texto definido pelo usuário.
  - É utilizado para equipamentos RDS não suportados diretamente pelo AUDICOM.
  - Se a aplicação do hardware suporta a leitura de um arquivo externo, esta opção permite a vinculação.
  - A localização suporta formato **UNC**.
