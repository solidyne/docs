  <Accordion title="GPIO, intercom e auxiliares">
    | Item | Especificação |
    | --- | --- |
    | Luz "No Ar" | 12 VCC / 250 mA |

    | GPO | 7 saídas (coletor aberto; \+24 V @ 250 mA) |

  <Accordion title="Arquitetura e processamento de áudio">
    | Item | Especificação |
    | --- | --- |
    | Buses | 4 estéreo (PGM, REC, AUX, VBUS) |
    | Bus PFL | 1 (CUE) |
    | Resolução / SR | 24 bits @ 48 kHz |
    | Processamento | DSP em todas as entradas (fixo por entrada) |
    | Processos | EQ paramétrico 3 bandas, gate, compressor/limitador, de-esser |
    | Mix-minus / talkback | Envios automáticos para TELCO |
    | Latência | 1,2 ms (linha→saída analógica) / 1,4 ms (MIC→saída com EQ\+Dinâmica\+De-Esser) |
  </Accordion>
