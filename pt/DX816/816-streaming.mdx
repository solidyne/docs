## Módulo de streaming AoIP

As consolas equipadas com o opcional **/AoIP** podem incluir até **dois módulos para serviços de streaming de áudio**:

- Com **1 módulo**: enviam sinal de **PGM**.
- Com **2 módulos**: enviam **PGM e REC**.

**Aplicações de AoIP:**

- **Enlace com planta transmissora** (STL) utilizando receptores dedicados (Solidyne **ADA102**,

  processadores Solidyne **542 APC /AoIP**).
- Enlace direto com outra consola /AoIP (Solidyne **2600**, séries **DX** ou **UNIDEX**).
- Envio do sinal PGM/REC para um computador remoto na rede, decodificado com software padrão de streaming (ex: **VLC Player**).

### Alimentação de módulos AoIP

Os módulos AoIP são alimentados pelas conexões **USB**. Eles exigem que pelo menos um USB esteja conectado a um computador **ligado**.

<Warning>
  Se o console não estiver conectado a um computador USB, ou se este estiver desligado, o serviço IP **não estará disponível**.
</Warning>

<Note>
  Na ausência de um computador, é possível conectar uma **fonte de alimentação USB padrão (5V)** para energizar os módulos AoIP.
</Note>
