Os módulos **/AoIP** vêm de fábrica com firmware modelo **STL**, projetado para **link estúdio-transmissor**.

É possível atualizar o firmware para:

- Instalar uma **nova versão STL**.
- Mudar de **STL** para **Upstreamer**, para conexão direta à Internet com servidores de streaming tipo **Shoutcast/Icecast**.
