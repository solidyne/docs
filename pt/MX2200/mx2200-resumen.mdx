## Descrição

O codec-mixer **Solidyne MX2200** é uma unidade compacta para trabalho em campo e transmissões externas. Gerencia múltiplas tecnologias de conexão com operação simples e intuitiva.

Pode ser configurado através de sua **tela integrada** ou, quando conectado à rede, de um computador usando um navegador web.

É alimentado por uma fonte chaveada de **12 VCC / 1,25 A**, e sua bateria interna fornece até **4 horas de autonomia**.

## Enlace e transmissão

**Tipos de enlace:**

1. Ethernet via cabo
2. Wi-Fi (usando antena externa)
3. 4G LTE (usando modem USB)
4. Bluetooth (para um telefone celular)
5. Linha telefônica fixa (POTS)

**Métodos de transmissão:**

1. VoIP através de chamadas SIP (full duplex)
2. Streaming para servidores Icecast ou Shoutcast
3. Streaming RTP
4. Áudio via telefonia móvel
5. Chamada telefônica convencional

## Gravação

O MX2200 permite gravar mais de **100 horas** em sua memória interna, em formato **MP3**.\
É possível estabelecer corte automático por tempo.

Os arquivos gravados são acessados conectando o console à rede local e usando um computador com qualquer navegador web.

## Entradas de áudio

**Microfones**\
Três faders de 100 mm controlam até 4 microfones dinâmicos.\
O fader #3 controla as entradas **MIC 3** e **MIC 4**. Quando ambas as entradas são usadas, é recomendável utilizar microfones da mesma marca e modelo.

As diferenças de nível entre diferentes vozes são parcialmente compensadas através de um **compressor interno**, que atua sobre a mixagem dos microfones.

**Linhas**\
O MX2200 possui duas entradas estéreo para sinais de linha:

- Uma entrada analógica (conector TRS)
- Porta USB para conexão direta a um computador

## Conexões telefônicas

O equipamento gerencia:

- Uma linha telefônica convencional
- Um telefone celular vinculado via Bluetooth

Estas comunicações podem ser utilizadas **como enlace principal** com o estúdio ou como backup da conexão de dados (streaming ou SIP).\
Quando não são usadas como enlace, podem ser integradas à mixagem do programa.

<Warning>
  **Atenção!** A CONFERÊNCIA entre a linha telefônica e Bluetooth **não é suportada**.\
  É possível colocar ambas as linhas no ar, mas não se ouvirão entre si nem poderão dialogar.
</Warning>

## Monitoramento e prévias

- Podem ser conectados até **4 fones de ouvido** no painel traseiro. Existe um único ajuste de volume geral. Para obter níveis diferenciados, recomenda-se usar um distribuidor de fones externo ou fones com controle de volume incorporado.
- Um **microfone de ordens integrado** permite falar em privado para o celular vinculado ou linha telefônica, ou enviar mensagens para os fones de ouvido.
- O circuito de monitoramento (**CUE**) permite escutar sinais antes de enviá-los à mixagem do programa, por exemplo para dialogar com as linhas.

**Luz de Ar (Tally light)**\
Se uma luz de ar for conectada, a saída **HEADPHONES #1** se transforma em saída para alto-falantes, já que é silenciada quando os microfones estão ativos, evitando microfonia.

## Saídas

O MX2200 oferece:

- Saída analógica estéreo
- Envio estéreo via USB
- Um CODEC integrado que permite gerar simultaneamente:
  - Uma comunicação SIP (full duplex)
  - Dois streamings: um para um endereço IP via RTP e outro para um servidor Icecast/Shoutcast

A mixagem principal inclui um **ponto de inserção estéreo (analógico)** para conectar um processador de áudio externo.
