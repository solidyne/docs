A atualização do sistema é realizada através da interface web da console:

<Steps>
  <Step title="Acessar o sistema de atualização">
    Digite no navegador o endereço IP da console seguido do subpath **/update**.\
    Exemplo: `http://192.168.0.121/update`
  </Step>
  <Step title="Verificar pacotes disponíveis">
    Na tela são exibidos os pacotes de software instalados e, se houver, as versões mais recentes disponíveis para atualização.
  </Step>
  <Step title="Baixar atualização">
    Clique em **Download** para baixar a versão mais recente.\
    O pacote é salvo na **memória interna da console**.
  </Step>
  <Step title="Instalar atualização">
    Uma vez finalizado o download, a opção **Install** é habilitada.\
    Clique para iniciar o processo de instalação.
  </Step>
</Steps>

<Info>
  - A console permite a atualização inclusive via **Wi-Fi**.
  - Como o pacote é armazenado localmente antes de ser instalado, a console poderá completar a instalação de forma autônoma.
  - A seção **Hardware Info** mostra informações do **hardware** como a quantidade de ciclos de carga da bateria.
</Info>
