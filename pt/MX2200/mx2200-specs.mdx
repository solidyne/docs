<Accordion title="CODEC – LINK">
  - **SIP:** Opus 48 kHz · Speex 32k / 16k / 8k · G729 · G722
  - **Streaming Icecast/Shoutcast:** MP3 · Opus 64–320 kbps
  - **Streaming RTP:** PCM 32–44.1–48 kHz · MP3 64–320 kbps · Opus 64–320 kbps
</Accordion>

<Accordion title="Conectividade">
  - Bluetooth 4.0
  - Gigabit Ethernet RJ45
  - Wi-Fi 2.4G / 5.8G (antena USB incluída)
  - Modem 4G USB externo (não incluído)
</Accordion>

<Accordion title="Entradas">
  - 4 × MIC dinâmicos (XLR)
    - Fader #3 controla MIC3 \+ MIC4
    - Nível: –70 a –30 dBu
  - 1 × USB áudio do PC (Fader #4)
    - Sem USB, Fader #4 comuta para linha estéreo analógica
  - 1 × Linha estéreo (Telefone/Linha ou Fader #4)
    - Nível –10 / \+20 dBu
  - 1 × Linha telefônica POTS (RJ11)
    - VoIP através de adaptador FXS externo (não incluído)
  - 1 × Smartphone via Bluetooth 4.0 (50 Hz – 8 kHz voz)
  - Insert PGM estéreo para processador externo (\+4 dBu)
</Accordion>

<Accordion title="Saídas">
  - Saída de Programa Estéreo: Analógica, não balanceada \+4 dBu
    - Máx. \+14 dBu · Z \>5 kΩ
  - USB Programa Estéreo para PC
  - Saída MIC Mixer para gravação em PC (–10 dBu / 10kΩ)
  - 4 × saídas de fones de ouvido (16–600 Ω)
    - Nível \+5 dBu / 50 Ω
  - Alto-falantes monitores (substitui saída Headphone #1 se houver luz de ar conectada)
  - Saída 12 V / 120 mA para luz de ar (Tally light)
</Accordion>

<Accordion title="Gravação">
  - Gravação interna em formato MP3 estéreo
  - Mais de 100 horas de armazenamento (@192 kbps)
  - Resolução 16 bits / 48 kHz
  - Bitrate configurável: 64 a 320 kbps
  - Nível nominal –10 dBFS · Faixa dinâmica 80 dB
</Accordion>

<Accordion title="Desempenho de Áudio">
  - **RTP PCM 48 kHz:**
    - Resposta: 20–18.000 Hz ±1 dB
    - Ruído / Faixa dinâmica \>85 dBA
    - Distorção menor que 0,01 % THD
  - **MIC para PGM Out:**
    - 20–15.000 Hz ±1 dB (com anti-pop –3 dB @30 Hz)
    - 20–20.000 Hz ±2 dB
  - **USB para PGM Out:**
    - 20–20.000 Hz ±0,5 dB
  - **Ruído:**
    - MIC EIN –120 dBA
    - Line in S/N = 85 dBA
  - **Compressor MIC:**
    - Limiar 0 VU
    - Compressão até 20 dB
    - Attack 5 ms · Release 200 ms
  - **Distorção:**
    - USB In → USB PGM Out menor que 0,02 % THD @1 kHz
    - MIC –50 dBu → PGM Out menor que 0,1 % THD
</Accordion>

<Accordion title="Híbrida Telefônica">
  - Resposta: 300–3400 Hz ±1 dB
  - Rejeição: \>40 dB @1 kHz (ajuste NULL traseiro)
  - Relação S/N \>60 dBA
  - Discador DTMF incorporado
</Accordion>

<Accordion title="Intercom">
  - Microfone interno de ordens
  - Comunicação privada com estúdio, fones de ouvido, linha telefônica e celular
</Accordion>

<Accordion title="Fonte de Alimentação">
  - Fonte chaveada externa 90–240 V AC 50/60 Hz → 12 V / 1,25 A
  - Bateria interna Li-Ion 2×3,7 V / 2500 mAh (com carregador interno)
  - Autonomia aprox. 4 h (Wi-Fi ligado, Bluetooth desligado)
  - Compatível com Power Bank externo 12 V DC / 1000 mAh (≈15 h de operação)
</Accordion>

<Accordion title="Insert de Dispositivos Externos">
  - Insert PGM estéreo para processador externo
  - Conector TRS ¼" "Stream In" comutado
  - Substitui sinal interno de PGM no streaming e USB
  - Nível \+4 dBu
</Accordion>

  - Altura frontal: 30 mm
  - Altura traseira: 95 mm
  - Peso: 2,4 kg
</Accordion>
