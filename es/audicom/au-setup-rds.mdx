### Uso

- A medida que el **programa de Aire** reproduce títulos, notifica al módulo RDS.
- El módulo accede a la base de datos para obtener la información necesaria.

Funciones disponibles:

- **Mensajes en tiempo real**: escriba el texto en el campo _Flash Msg_ y pulse **[Flash]**.
- **Anuncio de tráfico**:
  - Active el casillero _Send as traffic announce_.
  - Algunos receptores RDS muestran simultáneamente la información principal y el anuncio de tráfico; otros alternan entre ambas.
- **Configuración diferenciada**: es posible definir información distinta para música, bloques comerciales u otros bloques.
- **Exportar info a**:
  - Vuelca la información generada por RDS a un archivo de texto definido por el usuario.
  - Se utiliza para equipos RDS no soportados directamente por AUDICOM.
  - Si la aplicación del hardware soporta lectura de un archivo externo, esta opción permite la vinculación.
  - La ubicación soporta formato **UNC**.
