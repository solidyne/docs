## Conexión a la red local

Todas las opciones de configuración y ajustes avanzados del procesador **542APC** se realizan desde la **interfaz de control WEB**.

Para acceder:

1. Conecte el equipo a una boca del **router o switch** de la red local usando el **puerto Ethernet Control**.

   <Warning>
     No confundir el **puerto Ethernet de Control** con el **puerto Ethernet AoIP** (modelos con opción Streamer/AoIP).
   </Warning>
2. La **dirección IP asignada** se muestra en la pantalla OLED del frente del equipo.

   <discovery />
     
   <Info>
     De forma predeterminada, el 542APC trabaja con IP dinámica (DHCP activado):

     El router asigna automáticamente una dirección IP libre.

     Esta dirección puede cambiar al reiniciar el equipo.

     Es posible asignar una dirección\*\*IP estática \*\* desde la pantalla de sistema para evitar cambios de IP.
   </Info>
3. Usando una computadora conectada a la misma LAN que el procesador, ingresar la dirección IP en un **navegador web**  (recomendado Google Chrome).

   <Info>
     Se recomienda que la computadora esté conectada a la LAN por **cable Ethernet**.\
     En algunos casos, una PC conectada por Wi-Fi podría no tener permisos para acceder a dispositivos cableados de la misma red.
   </Info>

## Acceso remoto vía Internet

### Unidades con módulo STREAMER AoIP

En unidades equipadas con la expansión _STREAMER AoIP_ (con número \_serie AD \_y posteriores) el acceso al procesador vía Internet está simplificado a través de un servicio que genera un enlace fijo.

<Icon icon="globe" iconType="solid" color="#ff6c00" size={18} />  Ver Acceso remoto simplificado → Equipos con servicios AoIP

### Unidades sin STREAMER AoIP

Si el equipo no cuenta con el **servicio simplificado**, se puede acceder desde fuera de la LAN de dos maneras:

#### 1. Usando un software de Escritorio Remoto

Un método sencillo, que no requiere mayores configuraciones, es usar un software de **escritorio remoto** (TeamViewer, Anydesk, Parsec, etc.) para acceder a una computadora de la LAN conectada por Ethernet al procesador.

- Desde esa computadora local, abra un navegador web y acceda a la **interfaz web del 542APC**.
- De esta forma es posible configurar remotamente el procesador e incluso instalar actualizaciones de firmware sin riesgos.
- La comunicación entre la PC y el procesador ocurre dentro de la **red local**.
- Internet solo interviene en el control remoto de la computadora.

<Tip>
  Este método es el más seguro y sencillo para soporte remoto, ya que no requiere modificar configuraciones de red.
</Tip>

#### 2. Configurar acceso externo a la IP del equipo

La segunda opción consiste en instalar el procesador con una **dirección IP estática** y configurar la red local para permitir acceso externo a esa dirección.

Esto implica:

- Configuración de **re-direccionamiento de puertos**.
- Contratación de **IP públicas estáticas** y/o uso de servicios de **DNS dinámico**.

<Warning>
  Este procedimiento requiere conocimientos avanzados de redes.\
  Se recomienda consultar a un técnico especializado en administración de redes.
</Warning>
