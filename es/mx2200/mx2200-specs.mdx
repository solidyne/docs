<Accordion title="CODEC – LINK">
  - **SIP:** Opus 48 kHz · Speex 32k / 16k / 8k · G729 · G722
  - **Streaming Icecast/Shoutcast:** MP3 · Opus 64–320 kbps
  - **Streaming RTP:** PCM 32–44.1–48 kHz · MP3 64–320 kbps · Opus 64–320 kbps
</Accordion>

<Accordion title="Connectivity">
  - Bluetooth 4.0
  - Gigabit Ethernet RJ45
  - Wi-Fi 2.4G / 5.8G (antena USB incluida)
  - Módem 4G USB externo (no incluido)
</Accordion>

<Accordion title="Inputs">
  - 4 × MIC dinámicos (XLR)
    - Fader #3 maneja MIC3 \+ MIC4
    - Nivel: –70 a –30 dBu
  - 1 × USB audio desde PC (Fader #4)
    - Si no hay USB, Fader #4 conmuta a línea estéreo analógica
  - 1 × Línea estéreo (Telephone/Line o Fader #4)
    - Nivel –10 / \+20 dBu
  - 1 × Línea telefónica POTS (RJ11)
    - VoIP mediante adaptador FXS externo (no incluido)
  - 1 × Smartphone por Bluetooth 4.0 (50 Hz – 8 kHz voz)
  - Insert PGM estéreo para procesador externo (\+4 dBu)
</Accordion>

<Accordion title="Outputs">
  - Stereo Program Out: Analógico, desbalanceado \+4 dBu
    - Máx. \+14 dBu · Z \>5 kΩ
  - USB Stereo Program a PC
  - MIC Mixer Out para grabación en PC (–10 dBu / 10kΩ)
  - 4 × salidas de auriculares (16–600 Ω)
    - Nivel \+5 dBu / 50 Ω
  - Parlantes monitores (reemplaza salida Headphone #1 si hay luz de aire conectada)
  - Salida 12 V / 120 mA para luz de aire (Tally light)
</Accordion>

<Accordion title="Recording">
  - Grabación interna en formato MP3 estéreo
  - Más de 100 horas de almacenamiento (@192 kbps)
  - Resolución 16 bits / 48 kHz
  - Bitrate configurable: 64 a 320 kbps
  - Nivel nominal –10 dBFS · Rango dinámico 80 dB
</Accordion>

<Accordion title="Audio Performance">
  - **RTP PCM 48 kHz:**
    - Respuesta: 20–18.000 Hz ±1 dB
    - Ruido / Rango dinámico \>85 dBA
    - Distorsión menor a 0,01 % THD
  - **MIC a PGM Out:**
    - 20–15.000 Hz ±1 dB (con anti-pop –3 dB @30 Hz)
    - 20–20.000 Hz ±2 dB
  - **USB a PGM Out:**
    - 20–20.000 Hz ±0,5 dB
  - **Ruido:**
    - MIC EIN –120 dBA
    - Line in S/N = 85 dBA
  - **Compresor MIC:**
    - Umbral 0 VU
    - Compresión hasta 20 dB
    - Attack 5 ms · Release 200 ms
  - **Distorsión:**
    - USB In → USB PGM Out menor a 0,02 % THD @1 kHz
    - MIC –50 dBu → PGM Out menor a 0,1 % THD
</Accordion>

<Accordion title="Phone Hybrid">
  - Respuesta: 300–3400 Hz ±1 dB
  - Rechazo: \>40 dB @1 kHz (ajuste NULL trasero)
  - Relación S/N \>60 dBA
  - Dial DTMF incorporado
</Accordion>

<Accordion title="Intercom">
  - Micrófono interno de órdenes
  - Comunicación privada con estudio, auriculares, línea telefónica y celular
</Accordion>

<Accordion title="Power Supply">
  - Fuente conmutada externa 90–240 V AC 50/60 Hz → 12 V / 1,25 A
  - Batería interna Li-Ion 2×3,7 V / 2500 mAh (con cargador interno)
  - Autonomía aprox. 4 h (Wi-Fi encendido, Bluetooth apagado)
  - Compatible con Power Bank externo 12 V DC / 1000 mAh (≈15 h de operación)
</Accordion>

<Accordion title="Insert External Devices">
  - Insert PGM estéreo para procesador externo
  - Conector TRS ¼” “Stream In” conmutado
  - Reemplaza señal interna de PGM en streaming y USB
  - Nivel \+4 dBu
</Accordion>

  - Profundidad: 284 mm
  - Altura frontal: 30 mm
  - Altura trasera: 95 mm
  - Peso: 2,4 kg
</Accordion>
