Multi-Discovery permite encontrar en una LAN distintos tipos de interfaces Solidyne conectadas.

**Instalación**

Descar el archivo e instalar siguiendo las instrucciones en pantalla: https://www.solidynepro.com/DW/IP.exe

**Uso de Multi-Discovery**

1. El menú _"Protocolo"_ despliega opciones para encontrar distintos tipos de interfaces:
   - **IP Audio Module:** Servicios de streaming
   - **Solidyne:** Interfaces de control remoto
2. En "_Network Interface", s_eleccionar la placa de red (si existiera más de una).
3. Pulsar _"Discover..."_ para iniciar la búsqueda. 
4. Haciendo doble clic sobre un ítem en la lista, se abrirá elñ navegador predeterminado mostrando la interfaz web del dispositivo.
