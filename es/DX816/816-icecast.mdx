Los módulos **/AoIP** vienen de fábrica con firmware modelo **STL**, diseñado para **enlace estudio–planta**.

Es posible actualizar el firmware para:

- Instalar una **nueva versión STL**.
- Cambiar de **STL** a **Upstreamer**, para conexión directa a Internet con servidores de streaming tipo **Shoutcast/Icecast**.
