## Conector de salida STUDIO MONITOR

El conector de salida **STUDIO MONITOR** transporta las señales de monitoreo hacia el estudio principal (Estudio A).

Las bahías de monitoreo **Solidyne StudioBox HD3 y HD5** se conectan directamente a esta salida mediante un cable **RJ45–RJ45 STP**, con conexión **pin a pin**.

Cuando no se utiliza una StudioBox, la consola incluye un cable adaptador **RJ45 a conectores de audio**, que permite conectar directamente altavoces activos y distribuidores de auriculares en el estudio.

<Warning>
  La salida **STUDIO MONITOR** no debe conectarse utilizando cables diferentes al original ni mediante tramos adaptadores RJ45 genéricos. Esto puede provocar fallos o daño en la salida Studio Monitor.
</Warning>

Las señales presentes en el conector **STUDIO MONITOR** son:

| **PIN** | **SEÑAL**                       |
| :-----: | ------------------------------- |
| 1       | Auriculares Izquierdo (\+4 dBu) |
| 2       | Auriculares Derecho             |
| 3       | GND                             |
| 4       | ON-AIR (\+14 V)                 |
| 5       | Talkback a Cabina de Control    |
| 6       | VCC \+14 V                      |
| 7       | Parlantes Izquierdo (\+4 dBu)   |
| 8       | Parlantes Derecho               |
|         | \*Chasis = GND                  |
