<Tip>
  Cada entrada se puede asignar a cualquiera de los canales físicos o virtuales.
</Tip>

## <Icon icon="arrow-right-to-line" iconType="solid" color="#008dff" size='22' />  Entradas para micrófonos

Las entradas balanceadas para micrófonos soportan con alimentación _phantom_ **48 V.** La tensión se activa/desactiva para cada canal desde la pantalla.

Cada entrada dispone de un **microinterruptor que permite reducir la ganancia del preamplificador en –12 dB**. El interruptor en posición hacia abajo **activa la atenuación**.

<Info>
  En UX18, la entrada MIC-5 es de nivel fijo.
</Info>


## <Icon icon="arrow-right-to-line" iconType="solid" color="#6f07ab" size='22' />  Entradas tipo UNIDEX

Las entradas tipo **UNIDEX** son entradas especiales que, además de audio, transportan **alimentación PoE** y canales de **comunicación de control** para operar los **adaptadores externos** Solidyne **UDX**.

Normalmente se utilizan para conectar **adaptadores UDX** que permite agregar micrófonos, líneas telefónicas o teléfonos celulares.

Las entradas UNIDEX también **se pueden usar como entradas balanceadas** convencionales. Cada conector de entrada UNIDEX contiene la señales balanceadas izquierda y derecha.

## <Icon icon="arrow-right-to-line" iconType="solid" color="#008dff" size='22' />  Entradas analógicas balanceadas

- Cada entrada utiliza un **único cable** STP que transporta las señales balanceadas **izquierda y derecha**.
- En el extremo, el RJ45 se adapta al tipo de conector de audio necesario (XLR, TRS, RCA).

Para conectar una fuente en **modo desbalanceado**, se debe utilizar un **cable adaptador adecuado**, conectando:

- El terminal de señal **(–)** a la **malla** del cable.
- El terminal de señal **(\+)** como el **vivo** de la señal.

## <Icon icon="arrow-right-to-line" iconType="solid" color="#008dff" size='22' />  Entradas digitales AES3

- Se conectan mediante cable **RJ45–RJ45 CAT5**, utilizando el adaptador **SOL-F3/XLRF**.
- Para facilitar la compatibilidad, emplean el **mismo pin-out** que las entradas balanceadas estéreo, aunque **solo se utiliza el canal izquierdo** para señal digital.
- Soportan señales **S/PDIF**.

<Note>
  Cuando se habilita la función de bajada de streaming, el audio decodificado reemplaza internamente a la entrada **AES1**. El conector **AES-1** del panel trasero queda automáticamente desconectado.
</Note>

## Entrada para sintonizador externo

La consola dispone de una **entrada estéreo para sintonizador externo**, mediante conector **TRS de 1/4”**. Esta entrada permite monitorear directamente la señal que está saliendo al aire.

<Tip>
  En FM, es importante que tanto en los estudios como en la cabina de control, el monitoreo se realice con la señal **de aire procesada**, no desde el bus PGM directo. Esto permite detectar errores reales de emisión y brinda una escucha más confortable, sin saltos de volumen que pueden ocurrir en la señal directa.
</Tip>
