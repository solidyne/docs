El puerto **USB** incorporado permite el intercambio de señales estéreo entre la consola y un computador con Windows 10.

- <Icon icon="arrow-right-to-line" iconType="solid" color="#008dff" size={22} /> Proporciona **4 dispositivos de reproducción estéreo** (de PC hacia "entradas USB" en la consola).
- <Icon icon="arrow-left-from-line" iconType="solid" color="#00860b" size={22} /> Proporciona **4 dispositivos de grabación estéreo** ("salidas USB" de la consola hacia la PC).

Es decir, la computadora puede **enviar 4 señales estéreo** a la consola, y **recibir otras 4 señales estéreo** desde ella.

<Info>
  Aunque USB es técnicamente un puerto de datos, en este contexto lo referimos como **entradas y salidas de audio USB**.

  - Las “**salidas USB**” de la consola corresponden a **dispositivos USB de grabaciópn** en Windows.
  - Las “**entradas USB**” de la consola corresponden a \*\*dispositivos USB de reproducción \*\*en Windows.

    Tener esto en mente es fundamental para comprender correctamente el flujo de señales.
</Info>

La asignación de señales hacia las salidas USB (es decir, los dispositivos de grabación USB en Windows) se configura desde la consola en la pantalla `SETTINGS → OUTPUTS`

<Warning>
  La siguiente descripción se refiere al **conector USB incorporado** en el Panel Trasero.\
  Para instalar un **adaptador externo UDX-USB**, consultar la sección: **Adaptadores externos UNIDEX**
</Warning>

## Instalación del controlador USB

UX24 requiere un controlador USB propio para establecer comunicación con el sistema operativo Windows.

### Pasos para la instalación:

1. Conectar el puerto USB de la consola al PC utilizando el cable **USB A/B** suministrado.

<Danger>
  El uso de cables de **mala calidad o dañados** puede provocar que el computador no reconozca correctamente el dispositivo USB o lo identifique con fallos.
</Danger>

2. Descargar el controlador desde:\
   https://solidynepro.com/DW/Solidyne_UsbAudio.zip
3. Ejecutar el instalador y seguir las instrucciones.

### Dispositivos USB en Windows

Una vez instalado el driver correctamente, Windows mostrará:

Dispositivos de reproducción (salidas desde la PC hacia UX24):

- **Solidyne USB OUT 1 →** Entrada USB 1 (UX24)
- **Solidyne USB OUT 2 →** Entrada USB 2 (UX24)
- **Solidyne USB OUT 3 →** Entrada USB 3 (UX24)
- **Solidyne USB OUT 4 →** Entrada USB 4 (UX24)

Dispositivos de grabación (envíos desde UX24 hacia la PC):

- **Solidyne USB IN 1 ←** Salida USB 1 (UX24)
- **Solidyne USB IN 2 ←** Salida USB 2 (UX24)
- **Solidyne USB IN 3 ←** Salida USB 3 (UX24)
- **Solidyne USB IN 4 ←** Salida USB 4 (UX24)

Estas salidas se asignan desde la pantalla en: `SETTINGS → OUTPUTS`

## Compatibilidad con ASIO 3.0

La consola **UX24 soporta ASIO 3.0**, lo que permite compartir simultáneamente el uso de los canales USB entre distintas aplicaciones.

### Ejemplo de aplicación avanzada:

- El software de automatización reproduce música por el canal **USB 1**.
- Otro software de procesamiento (como un host de efectos) toma el canal **USB 2** de retorno desde la consola y aplica **efectos VST** en tiempo real a las voces de micrófono.
