  <Accordion title="GPIO, intercom y auxiliares">
    | Función                 | Especificación                                                       |
    | ----------------------- | -------------------------------------------------------------------- |

  <Accordion title="Arquitectura y procesado de audio">
    | Parámetro            | Valor                                                                                                            |
    | -------------------- | ---------------------------------------------------------------------------------------------------------------- |
    | Buses de mezcla      | 4 buses estéreo (PGM, REC, AUX con asignación directa desde la superficie; VBUS asignable desde pantalla)        |
    | PFL                  | 1 bus PFL (CUE)                                                                                                  |
    | Resolución / SR      | 24 bits @ 48 kHz                                                                                                 |
    | Procesos DSP         | Fijos por entrada y según tipo: EQ paramétrico (3 o 4 bandas), compuerta de ruido, compresor/limitador, de-esser |
    | Mix-minus / talkback | Envíos automáticos (TELCO)                                                                                       |
    | Latencia             | 1.2 ms (línea analógica → salida analógica) / 1.4 ms (MIC con EQ + dinámica + de-esser)                          |
  </Accordion>
