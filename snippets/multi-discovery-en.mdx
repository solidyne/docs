Multi-Discovery allows you to find different Solidyne interfaces connected to the network. Once the device is detected, its current IP address is displayed.

### **Installing Multi-Discovery**

Download the file and install it following the on-screen instructions: [Multi-Discovery](/es/otros_docs/downloads)

### **Using Multi-Discovery**

1. The _"Protocol"_ menu displays options to find different types of interfaces:
   - **IP Audio Module:** Streaming services
   - **Solidyne:** Remote control interfaces
2. In **_"Network Interface"_**, select the network adapter (if more than one is available).
3. Click **_"Discover..."_** to start the search.
4. Double-clicking an item in the list will open the default web browser showing the device's web interface.
