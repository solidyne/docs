<Accordion title="I/O Connectors">
  - **1/4" (6.32mm) stereo jack**
  - **AUDIO OUT**
    - TIP = Audio A / Audio A+B @ +4 dBu
    - RING = Audio B @ +4 dBu
  - **AUDIO IN**
    - TIP = Audio return from console (Mix-Minus) -10 dBu ~ +10 dBu
    - RING = TELCO output to Solidyne 'EXT HYBRID'
</Accordion>

<Accordion title="Operation Modes">
  - **MODE A 'Solidyne Consoles'**
    - Cellphones A and B in conference with console's internal hybrids.
    - DIP-2,4,5,6 = ON
    - Uses 1.80m stereo cable (supplied) from AUDIO IN (BLT200) to EXT HYB connector on Solidyne console.
  - **MODE B 'All consoles; 1 line input'**
    - All consoles in conference mode. DIP-2,4,5 = ON
    - **AUDIO IN**: stereo cable (supplied), Tip = return from console / Ring = n/c
    - **AUDIO OUT**: Y-cable (supplied), Tip = console line input / Ring = n/c
  - **MODE C 'All consoles; 2 line inputs'**
    - Conference mode. DIP-1,4,5 = ON
    - **AUDIO IN**: stereo cable (supplied), Tip = return from console.
    - **AUDIO OUT**: Y-cable (supplied), Tip = Line A / Ring = Line B.
  - **MODE D 'Independent cellphones'**
    - No conference. DIP-1 = ON
    - **AUDIO IN**: stereo cable (supplied), Tip = return from console.
    - **AUDIO OUT**: Y-cable (supplied), Tip = Line A / Ring = Line B.
  - **MODE E '2 × BLT200' (4 cellphones)**
    - Two BLT200 units, each in A/B conference mode. DIP-2,4,5 = ON
    - Connection as per MODE B wiring: Line 1 (Cell 1+2) and Line 2 (Cell 3+4).
    - Connect the two return bus inputs in parallel using a Y-adapter.
</Accordion>

<Accordion title="CONTROL Button Lighting Codes">
  - **A – Cellphone on hold**: brief yellow light (1 s) with long off period (2 s).
  - **B – Cellphone in communication**: slow yellow light (2 s) with short off period (1 s).
    - Indicates ongoing interview or searching for a previously paired device.
    - Returns to mode "A" when communication is ended (short press).
  - **C – Pairing mode**: very fast yellow flashing.
  - **D – Incoming call**: solid blue light while phone is ringing.
</Accordion>

<Accordion title="Audio Performance">
  - Frequency response: 50 Hz – 8,000 Hz (full voice range, Bluetooth 4.0 / 5.0)
  - Total Harmonic Distortion (THD): < 0.2% (measured input → output in electronic circuits)
  - Signal-to-noise ratio: > 70 dBA (in all electronic circuits)
</Accordion>

<Accordion title="Power Supply">
  - External switching power supply
  - 110/220V AC → 12V DC / 1A
</Accordion>
