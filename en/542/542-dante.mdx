## Dante/AES67 Module

The DANTE/AES67 expansion provides 2x2 AoIP connectivity.\
DANTE connectivity is used in configurations where the processor is located in the station's studios.

- It connects to the Dante network switch via the RJ45 Ethernet AoIP.
- Automatically recognized by Dante Controller (Audinate®).

  <img
    src="/images/542/542-dante.jpg"
    alt="542 Dante Jp"
    title="542 Dante Jp"
    style={{ width:"51%" }}
  />
- Stereo output is configured from **AUDIO OUTPUTS → Ethernet AoIP**.
  - The source can be the processed signal or the FM receiver.

    <img
      src="/images/542/542-dante-out.png"
      alt="542 Dante Out Pn"
      title="542 Dante Out Pn"
      style={{ width:"36%" }}
    />
- Input is configured from **AUDIO INPUTS → Ethernet AoIP**.

  <img
    src="/images/542/542-dante-input.png"
    alt="542 Dante Input Pn"
    title="542 Dante Input Pn"

  />
