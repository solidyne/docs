## What is Modified

Successive updates of the same software model can:

- Optimize processes.
- Add new features.
- Improve or modify the graphical interface.
- Incorporate new processing presets.

The **version and model** are displayed:

- On the startup screen of the **front panel**.
- On the top line of the **WEB Control** screen.

The unit can be updated while in service.\
It may cause a **brief audio interruption** on-air (less than 2 seconds).

## 542 APC Standard Model

Units with firmware 2.0 or later, which do not have the STREAMER AoIP expansion, are updated with the following procedure:

 1. Ensure the computer accessing the processor has **Internet connection** and is on the **same LAN** as the 542APC.
 2. Access the **WEB Control** page by entering the **control IP** in a browser.\
    (See [**Network Access and Control**](/en/542/542-acceso-web)).
 3. Enter **SETUP** mode (top right) and select the **INFO** option from the side menu.
 4. The unit will automatically check for an available update and **generate a direct download link**.

    <img
      src="/images/542/542-firm20-INFO-update.png"
      alt="542 Firm20 INFO Update Pn"
      title="542 Firm20 INFO Update Pn"
      className="mr-auto"
      style={{ width:"47%" }}
    />
 5. Download the firmware update from the link.
 6. Download and install the Windows application **Solidyne Discovery & Update** (link in the SOFTWARE section of the INFO screen).
    - The installer will create a _Solidyne Audio Processor_ folder in the Windows Start menu.
 7. Run the **Discovery & Update** application.

    <img
      src="/images/542/542-discovery.png"
      alt="542 Discovery Pn"
      title="542 Discovery Pn"
      style={{ width:"71%" }}
    />
 8. Click **Discover...** → Solidyne units on the network will be listed.
 9. Select the **542APC** unit and click **Update Firmware**.
10. In the pop-up window, select the update file downloaded in _Step 5_ and click **OK**.
11. The application will overwrite the firmware of the 542APC.
12. The unit will remain operational; there will be a **brief audio interruption (less than 2 seconds)** upon restart.

<Danger>
  **Do not power off or disconnect the unit during the update.**\
  It could cause irreparable user damage.
</Danger>

<Warning>
  Avoid performing updates from computers connected via Wi-Fi.
</Warning>


### Remote Updates

<Warning>
  Do not perform updates if accessing the unit directly via **Internet (port forwarding)**.\
  Always use **remote desktop software** to connect to a PC on the processor's LAN and access the **local WEB Control** from there.
</Warning>
