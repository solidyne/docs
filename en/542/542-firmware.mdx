## What is modified

Successive software updates for the same model can:

- Optimize processes.
- Add new features.
- Improve or modify the graphical interface.
- Add new processing presets.

The **version and model** are displayed:

- On the **front panel** startup screen.
- In the top line of the **WEB Control** screen.

The unit can be updated while in service.\
It may cause a **brief audio interruption** on-air (less than 2 seconds).

## 542 APC standard model

Units with firmware 2.0 or later, without the STREAMER AoIP expansion, are updated using the following procedure:

 1. Ensure the computer accessing the processor has an **Internet connection** and is on the **same LAN** as the 542APC.
 2. Access the **WEB Control** page by entering the **control IP** in a browser.\
    (See [**Network access and control**](/es/542/542-acceso-web)).
 3. Enter **SETUP** mode (top right) and select **INFO** from the side menu.
 4. The unit will automatically check for available updates and **generate a direct download link**.

 5. Download the firmware update from the link.
 6. Download and install the **Solidyne Discovery & Update** Windows application (link in the INFO screen SOFTWARE section).
    - The installer will create a _Solidyne Audio Processor_ folder in the Windows Start menu.
 7. Run the **Discovery & Update** application.

10. In the popup window, select the update file downloaded in _Step 5_ and click **OK**.
11. The application will overwrite the 542APC firmware.
12. The unit will remain operational; there will be a **brief audio interruption (less than 2 sec.)** when restarting.

<Danger>
  **Do not power off or disconnect the unit during the update.**\

<Warning>
  Avoid performing updates from computers connected via Wi-Fi.
</Warning>

### Remote updates

<Warning>
  Do not perform updates if accessing the unit directly via **Internet (port forwarding)**.\
  Always use **remote desktop software** to connect to a PC on the processor's LAN and access the **local WEB Control** from there.
</Warning>
