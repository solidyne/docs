## Features Overview

The 542APC series processors support two types of network audio expansion modules.

<Warning>
  - These options are mutually exclusive, they cannot be installed together.
  - They can come factory-installed or be added after purchase.
  - They are only supported by units with **serial numbers** starting with AD or later.
</Warning>

### 1. STREAMER Option (542APC /AoIP)

Features:

1. Outgoing RTP streaming (studio-transmitter link).
2. Outgoing streaming to Icecast/Shoutcast servers.
3. Incoming stream decoding (point-to-point link with transmitter site).
4. Remote audio monitoring via web.
5. Simplified direct remote access through Internet, without requiring static IP or port forwarding.
6. Simplified firmware updates.

### 2. DANTE/AES67 Option (542APC /A67)

2x2 Dante interface, designed for AoIP network connection in studios.

1. Stereo output of processed signal.
2. Stereo input.

### Module Installation

Expansion modules are sold separately. Although installation is straightforward, it requires disconnecting and opening the unit, so it is recommended to be performed by qualified technical personnel.

<Info>
  Refer to the _Optional Features and Disassembly Manual_.
</Info>
