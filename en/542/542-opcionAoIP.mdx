## Feature Summary

The 542APC series processors support two types of expansions for audio handling over networks.

<Warning>
  - These options are mutually exclusive and cannot be installed together.
  - They can be factory-installed or added after purchase.
  - They are only supported by equipment with **serial numbers** starting with AD or later.
</Warning>

### 1. STREAMER Option (542APC /AoIP)

Features:

1. Generation of an outgoing RTP stream (studio-transmitter link).
2. Outgoing streaming to Icecast/Shoutcast servers.
3. Decoding of an incoming stream (point-to-point link with the transmitter).
4. Remote audio monitoring via web.
5. Simplified direct remote access over the internet, without the need for a static IP or port forwarding.
6. Simplified firmware updates.

### 2. DANTE/AES67 Option (542APC /A67)

Dante 2x2 interface, designed for AoIP network connections in studios.

1. Stereo output of processed signal.
2. Stereo input.

### Module Installation

Expansion modules are sold separately. Although installation is straightforward, it requires disconnecting and opening the equipment, so it is recommended to be performed by qualified technical personnel.

<Info>
  Refer to the _Optional and Disassembly Manual_.
</Info>
