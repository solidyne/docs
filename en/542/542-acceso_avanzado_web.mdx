## Initial MONITORING Screen

When accessing the processor via network, the web control interface displays **monitoring screens** that show the action and current status of the different stages of the equipment.

![542 Web Home Pn](/images/542/542-web-home.png)

- These screens **do not allow configuration modifications**.
- To edit parameters, it is necessary to activate **SETUP mode** by pressing the button located in the upper right corner of the window.

The monitoring screens are organized in sections that can be **expanded or collapsed** by clicking the icon (±):

1. General Monitoring (_fixed section_)
2. Presets and Sound Wizards
3. FM Receiver – Monitor Analyzer
4. Audio Inputs and Outputs
5. Time Graph

### _General Monitoring_ Section

The upper section is **fixed** and remains visible in all modes (MONITORING and SETUP).

Includes:

- Active **input level** (RMS and peaks in dBFS).
- **Process** action indicators:
  - Enhancers status (active/inactive).
  - AGC action (dB).
  - Compressors and limiters attenuation (dB).
  - Wideband clipper and limiter status (dB).
  - MPX clipper (dB).
- Audio **output levels**:
  - Dual peak and RMS level indicator.
  - Loudness levels according to EBU-R128 standard.

The top row displays:

- Equipment **IP** address.
- Signal source and headphone volume.
- Current user (for password access control).
- Button to switch to **SETUP** mode.

---

### _Preset and Sound Wizard_ Section

Shows the **current processing preset** and simplified **Wizard** controls, which allow quick customization of the sound character.

- The user can change presets and adjust basic parameters (brightness, bass, compression, loudness).
- With the **CLICK HERE TO EDIT** option, you access **SETUP** mode to save changes or create new presets.

---

### _FM Receiver – Monitor Analyzer_ Section

Displays the main parameters of FM transmission analysis. Advanced receiver configuration and detailed reports are available in **SETUP** mode, **FM Receiver** menu option.

---

### _Audio Inputs and Outputs_ Section

Shows level indicators for:

- Audio inputs.
- Audio outputs.
- **MPX1** and **MPX2** composite outputs.

---

### _Time Graph_ Section

Graph showing **modulation evolution over time**.\
Allows switching between:

- MPX level.
- Modulation measured by FM receiver.
- MPX power according to **ITU BS412**.

## Advanced Access to Configuration and Settings

The **SETUP** button, located at the top of the screen, allows **editing parameters and settings** of the **542APC** processor.

- Access to this mode can be **password** protected.
- Upon entry, a **drop-down side menu** appears on the left side of the screen, from where all adjustment and configuration instances are accessed.

<Info>
  - Whenever an audio processing value is modified, **SAVE PRESET** must be pressed to retain the change. The SAVE option will flash while there are unsaved changes.
  - **System configuration** changes are saved automatically.
  - The **active processing preset** is a system value: each time it is changed, the setting is automatically saved.
</Info>
