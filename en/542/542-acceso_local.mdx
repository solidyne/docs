## Front Panel Control

Essential aspects of the 542APC can be configured directly from the front panel.\
The unit features an **OLED display** and a **push-to-select control wheel**.

1. Turn the wheel to navigate options.
2. Press the wheel to access screen options.
3. Turn again to navigate options within that screen.
4. To modify an option:
   - Press the wheel on the option.
   - Turn to change its value or state.
   - Press again to confirm.
5. To exit, select **BACK** and confirm.

### Lock Screen (LOCK)

The **LOCK** screen is accessed by turning the wheel from the home screen.\
It enables/disables password-protected access.

<img
  src="/images/542/542-oled-lock.png"
  alt="542 Oled Lock Pn"
  title="542 Oled Lock Pn"
  style={{ width:"45%" }}
/>

- To **lock**, select LOCK and press.
- To **unlock**, press LOCK again.
  - If password is active, it will be requested (default: `1234`).

<Info>
  When the unit is locked, screens can be browsed but options cannot be modified.
</Info>

Shows signal level for all inputs.\
The active input is indicated by a small **arrow** to the right of the input number.\

#### Changing the Input (from front panel)

1. Turn the wheel until **INPUT** is displayed.
2. Press the wheel to change the active input.
3. Turn to select the input (example: `2:ANALOG2`).
4. Press to confirm.
5. The active input is indicated with an arrow.
6. Gain adjustment is done from **SETUP**.

   />

   <img
     src="/images/542/542-oled-sistema-setup.png"
     alt="542 Oled Sistema Setup Pn"
     title="542 Oled Sistema Setup Pn"
     style={{ width:"45%" }}
   />

#### Input Setup

- `INPUT SEL`: selects active input.
- `GAIN`: adjusts active input gain within **±14 dB** range.
  - +4 dBu level equals –18 dBfs (AES K18), with **18 dB headroom**.
  - Adjust gain so **0 VU peaks at console** fall between **–22 and –9 dBfs** on processor meter.

  <img
    src="/images/542/542-oled-setup-entradas.jpg"
    alt="542 Oled Setup Entradas Jp"
    title="542 Oled Setup Entradas Jp"
    style={{ width:"34%" }}
  />

To modify gain:

1. Turn wheel, select `IN` and press.
2. (Optional) To change input, turn wheel, select `INPUT SEL` and press. Turn until desired input appears and press to confirm.
3. Select `GAIN` and press.
4. Turn to adjust gain.
5. Press to confirm.
6. Select **BACK** to exit.
7. Turn to **BACK** in SETUP menu to finish.
8. Changes are saved automatically.

#### Outputs Setup

- `OUTPUT ADJ`: allows selecting an output.
- `GAIN`: adjusts selected output level, following similar procedure as inputs.
