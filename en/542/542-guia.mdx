<Steps>
  <Step title="Power Supply and Startup" stepNumber={1} titleSize="h3">
    Before plugging in the unit:

    - Check the **mains voltage selector** position on the rear panel (200/240V or 100/130V).
    - Use the supplied **three-pin Interlock cable**.
    - Ensure proper **ground connection**.
    - The unit has a **power switch** on the rear panel.

    <Danger>
      Never connect the unit without verifying the correct position of the voltage selector.
    </Danger>
  </Step>
  <Step title="Unit Control" stepNumber={2} titleSize="h3">
    **From the front panel:**

    The front panel includes a **control wheel**:

    - Turn the wheel to select options and change values.
    - Press the wheel to confirm a value or access an option.

    <Info>
      The display shows status menus, active inputs, gain settings and audio presets.
    </Info>
    **Remote control via WEB interface:**

    - Connect the 542APC **Ethernet** port to the local network using a standard UTP cable.
    - The unit comes with **DHCP** enabled: the router will automatically assign an IP address.
    - The IP address appears on the processor's **home screen**.
    - From a computer on the same network, open a web browser and enter the processor's IP address.
    - The **web remote control home screen** will be displayed.
    - To edit options, go to **ADJUST** mode in the top right corner of the screen.
  </Step>

  <Step title="Input Setup" stepNumber={4} titleSize="h3">
    - Analog inputs calibrated to **-18 dBfs** for nominal level of **+4 dBu**.
    - With console at 0 VU, input signal peaks should be between **-22 and -9 dBfs**.

    <Note>
      If needed, gain can be adjusted from the front panel.
    </Note>
    **Selecting an input (front panel)**

    1. Turn the wheel to the **INPUT (VU meters)** screen.
    2. Press the wheel to access **active input**.
    3. Turn the wheel and select desired input (e.g.: _2:ANALOG2_).
    4. Press to confirm.
    5. Active input is indicated by an arrow on the INPUT screen.

    **Changing input gain (front panel)**

     1. Turn the wheel to **SETUP MENU** and press to enter.
     2. Turn to select **IN** and press to enter input **SETUP**.
     3. Press to enter **INPUT SEL**.
     4. Turn to choose an input and confirm.
     5. Select **GAIN** and press.
     6. Adjust by turning the wheel (clockwise = more gain) verifying peaks are between **-22 and -9 dBFS**.
     7. Press to confirm.
     8. Turn to select **BACK** and confirm to exit menu.
     9. Turn again to BACK and confirm to exit SETUP menu.
    10. Changes are stored.
  </Step>

  <Step title="Tuner and Transmission Monitor" stepNumber={6} titleSize="h3">
    The 542APC includes:

    - **Internal tuner**
    - **Modulation meter**
    - **Calibration tone generator**

    **Reception Antenna**

    - Connect the telescopic antenna (included) to the **FM ANT (BNC)** connector.
    - Two antennas are supplied:
      - Telescopic (recommended at transmitter site).
      - Fixed micro-antenna (when field strength is excessive).
  </Step>
  <Step title="Modulation Level" stepNumber={7} titleSize="h3">
    **Calibration Tone**

    1. From a browser (preferably Chrome), enter the **unit's IP**.
    2. Click the **SETUP** button (top right).
    3. In the side menu, select **FM Output (MPX)**.
    4. Enable the **Calibrator**. Input audio is replaced by a sine wave.
       - Pilot: 9% â€“ RDS: 4% (factory values).

    **Modulation Adjustment**

    1. In Web Control, go to **FM Receiver/Analyzer**.
    2. Tune to the station frequency.
    3. Check level on the **modulation meter**.
    4. The **RxQTY** indicator should show **GOOD**.
    5. Adjust antenna orientation or type if reception is poor.
    6. Adjust **MPX input level at transmitter** until desired modulation is achieved (100%).
    7. Factory MPX output level: **5 Vpp** (adjustable via Web Control).
    8. Turn off calibrator: input audio will resume normal modulation.
  </Step>
  <Step title="Sound Settings" stepNumber={8} titleSize="h3">
    Listen to different factory PRESETS to set the sound that fits the station's needs. Tune in on a good audio system. You can listen using the 542APC's internal tuner (use good headphones).

    The unit includes **factory presets** and user memories:

    - 16 fixed presets (non-modifiable).
    - 16 user memories to create custom settings (from Web Control).

    ### Preset Selection (front panel)

    1. Turn the wheel to the **PROCESS** screen.
    2. Press to change current preset.
    3. Turn and select a new preset.
    4. Press to confirm. On-air sound will gradually change to new setting.

    ### Sound Wizard (Web Control)

    Allows modifying any preset with 4 basic controls:

    - Bright
    - Bass
    - Compression/Density
    - Loudness

    Additional stages (can be enabled/disabled):

    - Parametric EQ
    - Bass enhancer
    - Stereo enhancer
  </Step>
