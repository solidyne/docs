## File Manager Operation

<Note>
  This section covers the **File Manager** module (discontinued in Audicom Black and replaced by **Media Manager**). It is maintained for operational compatibility and users in transition.
</Note>

## Expanded Mode

- Clicking the **red arrow** opens a **second window** to the right of the module (similar to the main one).
- Used to **move or copy** files between folders, among other functions.
- Clicking on a file changes the **active window**, indicated by the **green arrow**.
- Playback buttons act on the file **selected in the active window**.

---

## Moving and Copying Files

<Steps>
  <Step title="Open second window">
    Enable **expanded mode** (red arrow) to have two panels.
  </Step>
  <Step title="Select files">
    - Multiple selection with **Ctrl + click** (one by one).
    - Continuous range with **Shift + click** (from first to last).
  </Step>
  <Step title="Choose destination">
    In the secondary window, open the **destination folder**.
  </Step>
  <Step title="Move">
    Drag the selection to the destination folder.\
    If the files were **registered in the Database**, their **location is automatically updated**.
  </Step>
  <Step title="Copy">
    Drag while **holding Ctrl**: you'll see a **(+) sign** indicating copy.
  </Step>
</Steps>

<Info>
  **autoDB Folders**

  - When moving/copying music to an **autoDB** folder:\
    a) If the file **was not** registered, it is **automatically added**.\
    b) If it **was already** registered, all info is kept **except "Style"**, which takes the **new folder name**.
  - From **autoDB â†’ non-autoDB** folder: Database **remains unchanged**.
  - The **autoDB** logic applies **only during move/copy**; it doesn't modify information afterward.
</Info>

<Warning>
  Avoid moving/renaming/deleting registered files **using Windows tools**:

  - If moved, location will be corrected when aired (AutoDJ/InfoMusic), but requires **exhaustive searches**.
  - If deleted or renamed, Database will mark the media as **\_missing**.
</Warning>
