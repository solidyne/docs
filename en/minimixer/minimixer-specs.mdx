<Accordion title="Technology">
  - Digital stereo line input to mixer via USB.
  - Digital program output via USB to PC for streaming.
  - Analog microphone inputs with high-performance operational amplifiers, EMI protected, and analog stereo PGM output.
  - Preamplifiers with feedback control: prevent signal saturation at any fader position.
</Accordion>

<Accordion title="Inputs">
  - **Microphones**: 4 × dynamic mic inputs.
    - Fader #3 includes dual preamp for **MIC 3 + MIC 4**.
    - Level: –75 to –25 dBu.
  - **USB**: digital input from PC (fader #4).
    - If USB is not in use, automatically switches to stereo line.
  - **Telephone**: landline to hybrid.
    - Compatible with SIP lines via external FXS adapter (not included).
  - **Mobile**: input via **Bluetooth 4.0** (30 Hz – 8 kHz, full-range voice quality).
  - **Stereo line**: switched between telephone/line or fader #4.
    - Level: –10 to +20 dBu.
  - **Insert**: input/output for audio processor or external devices.
    - Level: +4 dBu.
</Accordion>

<Accordion title="Outputs">
  - **Stereo program (PGM)**: analog, unbalanced.
    - Nominal level: +4 dBu.
    - Maximum level: +14 dBu.
    - Impedance: \> 5 kΩ.
  - **USB**: digital stereo program output to PC.
  - **Mic REC**: microphone mix output for PC recording.
    - Level: –10 dBu / 10 kΩ.
  - **Headphones**: 4 × outputs (16–32–600 Ω).
    - Level: +5 dBu / 50 Ω.
  - **Muted speakers**: replaces **Headphone #1** output (auto-mute with on-air light).
  - **Tally Light**: 12 V / 120 mA (LED).
  - **SmartCam (optional)**: USB output for camera control in Audicom & Close RadioTV.
</Accordion>

<Accordion title="Intercom">
  - Internal microphone in MiniMixer.
  - Private communication with studio, headphones, telephone line or mobile.
  - Telephone keypad for making calls without requiring an external phone.
</Accordion>

<Accordion title="Audio Performance">
  - **Frequency response (MIC → PGM OUT)**: 20 Hz – 20 kHz ±1 dB.
    - Includes anti-pop filter: –3 dB @ 30 Hz.
  - **Frequency response (USB → PGM OUT)**: 20 Hz – 20 kHz ±1 dB.
  - **Noise**:
    - MIC EIN = –120 dBA.
    - Line input signal-to-noise ratio: 90 dBA.
  - **Microphone compressor**:
    - Range: 0 – 20 dB.
    - Attack = 5 ms.
    - Release = 200 ms.
  - **Distortion**:
    - USB Line IN → USB PGM OUT: \< 0.02 % THD at 1 kHz.
    - Microphone (–50 dBu) → Analog PGM OUT: \< 0.1 % THD.
</Accordion>

<Accordion title="Telephone Hybrid">
  - Frequency response: 300 – 3,400 Hz ±1 dB.
  - Rejection: \> 40 dB at 1 kHz (adjusting NULL on rear panel).
  - Signal-to-noise ratio: \> 60 dBA.
</Accordion>

<Accordion title="Power Supply">
  - External switching power supply: 90–240 V / 50–60 Hz.
  - Output: 12 V / 1 A.
</Accordion>

<Accordion title="External Device Insertion">
  - Rear connector on PGM output for inserting processors or external devices (level +4 dBu).
  - Allows connecting a USB audio interface with **ASIO** support to use VST plug-ins for effects or multiband processing on PC.
</Accordion>

<Accordion title="Dimensions and Weight">
  - Width: 274 mm
  - Depth: 270 mm
  - Front height: 25 mm
  - Rear height: 71 mm
  - Weight: 1.6 kg
</Accordion>
