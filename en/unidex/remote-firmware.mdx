### Software Update

UX24 models **manufactured since 2023** can be updated via the web interface from a computer with Internet access.\
For earlier models without this feature, please contact: [**info@solidyne.ar**](mailto:info@solidyne.ar).

**Components that can be updated:**

- Mixing engine firmware (hardware).
- Graphical interface.
- Streaming tools.

#### Update Procedure

1. Connect the console's **Ethernet Streaming Out** and **Remote Control** ports to the local network.
2. Ensure the network has **Internet access**.
3. On the console's main screen, check the IP address assigned to **STREAMING IP**.

<img
  src="/images/ux24/unidex_streaming_ip.jpg"
  alt="UNIDEX Streaming IP"
  title="UNIDEX Streaming IP"
  style={{ width:"34%" }}
/>

<discovery />

 4. In the browser, enter the IP address followed by `/update`.\
    Example: `192.168.0.88/update`
 5. Press **Enter** to open the updates screen.

    <img
      src="/images/ux24/ux24-update-firmware.png"
      alt="UNIDEX Firmware Update"
      title="UNIDEX Firmware Update"
      style={{ width:"61%" }}
    />
 6. Click **CHECK FOR UPDATES**.
 7. If no updates are available, the system will inform you that it is up to date.
 8. If updates are available, a list with details will be displayed.
 9. Updates are **not installed automatically**.
10. To install an update, press **INSTALL**.

    <Warning>
      Depending on the type of update, the console may restart, interrupting the on-air signal for a few seconds.
    </Warning>
