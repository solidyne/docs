  <Accordion title="General features">
    | Parameter              | Specification                                                                               |
    | --------------------- | ------------------------------------------------------------------------------------------ |
    | Channels              | 24 channels; 12 physical faders expandable to 24 using UX-6 modules                        |
    | Virtual faders        | 12 (UX24 base) Operable from touchscreen or web interface                                  |
    | Remote control        | Internal web interface (HTML5) for remote operation via LAN                                |
    | Input distribution    | Configurable across 24 channels                                                            |
    | Scenes               | 12 user scenes (input configuration, channel assignment, processing settings)               |
    | Touchscreen          | 7" — 1024 × 600 pixels, multi-touch                                                        |
    | Channel displays     | OLED (name, input level, attenuation level, special indicators)                           |
  </Accordion>
  <Accordion title="Inputs">
    #### Analog inputs

    | Type                    | Quantity | Details                                    |
    | ----------------------- | -------: | ------------------------------------------ |
    | MIC                     | 4        | Switchable +48V phantom power per input    |
    | Balanced stereo line    | 4        | Level +4 dBu                              |

    #### Digital inputs

    | Type           | Quantity           | Details                                        |
    | -------------- | -----------------: | ---------------------------------------------- |
    | AES3 stereo    | 4                 | Digital inputs                                 |
    | USB stereo     | 4                 | Four playback devices on a computer           |
    | UNIDEX         | 16 mono (8 stereo) | Up to 16 additional mics or 8 TELCO devices   |

    | Processing | Details                     |
    | ---------- | --------------------------- |
    | DSP        | DSP processing on each input |
  </Accordion>
  <Accordion title="Outputs">
    #### Analog outputs

    | Type                                                   | Quantity    |
    | ------------------------------------------------------ | ----------- |
    | PGM balanced stereo                                    | 2           |
    | Assignable balanced stereo                             | 4           |
    | Unbalanced (mix-minus)                                 | 8           |
    | Control Room monitoring (speakers + headphones)         | 2 stereo    |
    | Studio monitoring (speakers + headphones + talkback)    | 2 stereo    |

    #### Digital outputs

    | Type           | Quantity                                    |
    | -------------- | ------------------------------------------- |
    | AES3 stereo    | 4                                           |
    | USB stereo     | 4 (recording devices on a computer)         |
    | AoIP streaming | 2 stereo (RTP, Icecast/Shoutcast)          |
  </Accordion>
  <Accordion title="GPIO, intercom and auxiliaries">
    | Function              | Specification                                                      |
    | -------------------- | ------------------------------------------------------------------ |
    | "On Air" lights      | 2 outputs (Studios A and B), 12 VDC, 250 mA                       |
    | GPI                  | 4 general purpose inputs (active high 3.3 ~ 15 V)                  |
    | GPO                  | 8 outputs (7 open collector; +24 V @ 250 mA; +1 × 12 VDC @ 250 mA) |
    | Air monitoring       | 1 unbalanced stereo input (external tuner)                         |
    | Talkback from studios| 1 input (on Studio Box connector)                                  |
    | Orders microphone    | Built-in (talk-back)                                               |
    | Intercom            | 4 assignable intercom buttons                                       |
    | TELCO intercom      | PTT (push-to-talk) on TELCO channels using CUE button              |
  </Accordion>
  <Accordion title="Audio architecture and processing">
    | Parameter            | Value                                                                                                          |
    | ------------------- | -------------------------------------------------------------------------------------------------------------- |
    | Mix buses           | 4 stereo buses (PGM, REC, AUX with direct surface assignment; VBUS assignable from screen)                     |
    | PFL                 | 1 PFL bus (CUE)                                                                                                |
    | Resolution / SR     | 24 bits @ 48 kHz                                                                                               |
    | DSP processes       | Fixed per input and by type: parametric EQ (3 or 4 bands), noise gate, compressor/limiter, de-esser           |
    | Mix-minus / talkback| Automatic sends (TELCO)                                                                                        |
    | Latency            | 1.2 ms (analog line → analog output) / 1.4 ms (MIC with EQ + dynamics + de-esser)                             |
  </Accordion>

    | Parameter              | Value                          |
    | --------------------- | ------------------------------ |

    | Frequency response    | 20 Hz – 20 kHz ± 0.5 dB        |
