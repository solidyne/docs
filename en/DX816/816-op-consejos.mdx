## Console Operational Verification

Basic procedure to confirm connection:

1. Use a signal source (microphone, playback device, etc.).
   - On channels 4 or 5, correctly select the input (BAL/LIN).
   - Common error: no signal due to incorrect input assignment.
2. Press the **CUE** button on the input channel.
3. Select **CUE** as monitoring source and adjust CUE level.
   - Audio should be heard in headphones (and monitors if assigned).

**To send signal on-air:**

1. Press **PGM** on the channel to route it to main output.
2. Open the channel fader to desired level.
   - Signal will be transmitted on-air and displayed on program VU meters.

<Warning>
  Gain must be adjusted on each channel (except USB) using panel gain presets, so that the normal working position of the master fader is **–15 dB (gray zone)**.
</Warning>

## Program Recording

Solidyne DX series consoles feature two stereo output buses:

- **PGM (Program)**: Signal sent on-air.
- **REC (Recording)**: Auxiliary signal, typically used for recording.

Each channel can be assigned to **PGM** or **REC** using assignment buttons, enabling recording while broadcasting different signals.

### USB Recording

The PGM and REC signals are available on the computer via **USB**.

On **<sup>Windows©</sup>**:

- **USB Recording Device 1** → corresponds to **USB-1**, PGM signal.
- **USB Recording Device 2** → corresponds to **USB-2**, REC signal.

On the **computer**:

- Configure the recording application to use the appropriate device.

On the **console**:

- In monitoring, assign **REC** to the Control Room or Studio speakers if you want the announcers to hear the recording.
- Remove from **PGM** and assign to **REC** the channels to be recorded (e.g., microphones).

  → These channels will NOT go on-air.
- The station remains on-air with another channel (e.g., Line-1 on PGM).

**Level Adjustment:**

- Check levels on the console's REC VU meters and in the software.
- Adjust from the console using the channel fader → **0 VU = –15 dBfs**.

<Danger>
  The **USB-1 and USB-2 recording devices have no software input level control**.\  
  Adjustment must always be done at the console.
</Danger>

**Monitoring:**

- Announcers can monitor via headphones or speakers.
- To play back the recording: assign to **REC** the channel corresponding to that PC.

<Warning>
  Avoid feedback loops on the PC. Mute in the recording software (or in Windows) any possible returns from the USB-2 (REC) input.
</Warning>

When finished, reset all controls to their original positions.

### Telephone Line Recording

The **TELCO** channel can be assigned to any of the outputs using the **PGM/REC** button.

Example: recording a telephone report while music is playing on-air.

- Assign the **hybrid** and a **microphone** to the **REC** output.
- Proceed with communication as in _Hybrid Channel_.
- Assign monitoring to **REC** to listen to the conversation.
