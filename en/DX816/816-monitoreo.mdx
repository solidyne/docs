## Control Room Monitoring

The \*\*CONTROL \*\*outputs feature **1/8" female TRS (mini-jack)** connectors for speakers and headphones. The speaker output operates at nominal **0 dBu** level, designed for active monitor connection.\
In the console's MONITOR section panel, you'll find the controls for signal routing and level adjustment for both speakers and headphones.

<Note>
  The **Control Speaker** output is automatically muted when **Talk-back** is activated, preventing feedback with the command microphone.
</Note>

---

## Monitoring in the Studio

The console delivers monitoring signals to the Studio via **STUDIO MONITOR OUT**, with an RJ45 connector. The connection is made using **STP RJ45-RJ45 (pin-to-pin)** cable to the optional **Solidyne Studiobox HD3/HD5** monitoring units.

<Danger>
  This output **must not be connected** with RJ45 cables used for audio outputs.\
  It is not compatible with the **StudioHub** standard.
</Danger>

Features:

- Studio speakers are **automatically muted** when microphones are opened.
- The headphone signal **is never muted**. It always receives the signal assigned on the console and the talk-back microphone audio.

For more details, see the **Studiobox HD3 / HD5 Monitoring Modules** section.

## Single Booth Monitoring

When the console is used in an integrated or single booth studio, it is recommended to connect the speakers to the **STUDIO MONITOR** output.

<Warning>
  The CONTROL speaker output **is not muted when microphones are turned on**, so there is a risk of feedback if the microphones and speakers are in the same room.
</Warning>

---

## Connection Without Studiobox Option

The console comes with a **Y adapter cable** (RJ45 to dual TRS jack) for connecting generic headphone distribution units and powered speakers.

- The **STUDIO MONITOR** output delivers signals at a fixed level.
- If Studiobox is not used, the speaker level is adjusted from the monitor's own controls.
- A dual-input headphone distribution/amplifier unit can be used to control the speaker volume from the console.
  - Input A → headphone signal.
  - Input B → speaker signal.

The speaker volume control is assigned to input B. This way, a single unit controls both headphones and studio monitors.
