## PC Connection

The DX series is designed to work with any communication software running on the computer. From videoconferencing applications like Zoom or Google Meets to SIP line receiving software such as Linphone.

The telecommunications channel allows sending a **differentiated return feed** (mix-minus) to the computer via USB-2, where the computer's own audio (USB-2 playback) is not mixed in, preventing feedback loops.

The computer can be added to conference calls managed by the Telecommunications channel.

<Note>
  While communication software typically has built-in cancellation mechanisms, these systems often become desynchronized, creating echoes and audio artifacts. A mix-minus send is essential for reliable communications.
</Note>

<Danger>
  ****USB-2 is non-operational if USB-1 is not connected to an active port.****\
  If USB-1 and USB-2 are connected to **two different PCs**, USB-2 **will not** function when the **USB-1 PC** is **powered off**.\
  (Typically USB-1 connects to the 24/7 on-air PC).
</Danger>

## Basic Configuration

<Steps>
  <Step title="Microphones">
    Assign microphone channels to **PGM** and **REC**.
  </Step>
  <Step title="USB-2 Channel">
    Assign to **PGM** and remove from **REC**.
  </Step>
  <Step title="Telco Channel">
    Set PGM-REC selection to **PGM**
  </Step>
  <Step title="Computer">
    Configure the application with **USB-2** audio device as input/output (microphone and playback).
  </Step>
  <Step title="ON-AIR">
    The application audio enters the console via the USB-2 Channel and goes on-air with microphone return (via REC).

    <Note>
      The application may be handling a single call or a multi-participant chat. This does not affect console operation.
    </Note>
  </Step>
</Steps>

## Phone/Cell Conference (Vox-IP)

<Steps>
  <Step title="Microphones">
    Assign microphone channels to **PGM** and **REC**.
  </Step>
  <Step title="USB-2 Channel">
    Assign to **PGM** and remove from **REC**.
  </Step>
  <Step title="Telco Channel">
    Set PGM-REC selection to **PGM**.\
    **Activate the VOX-IP button.** Phone/cell return is sent to PGM-2 via **REC** (visible on **REC** VU meter).
  </Step>
  <Step title="Conference">
    Raise the Telco fader to mix phone calls on-air; all participants become **interconnected**.
  </Step>
</Steps>

<Danger>
  For **return feed to computer (REC)**, do not assign **music beds**; only route **MICs**.\
  Priority/AGC systems in communication applications may **attenuate** remote voices when local audio (background music) is present.
</Danger>
