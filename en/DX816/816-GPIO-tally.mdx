## On-Air Light (Tally Light)

The **Tally Light** output delivers **12 VDC / 150 mA** when the **Microphone Master Button** is enabled.\
It allows connecting up to **two Solidyne LA100 signaling modules**.

Additionally, the console includes a **built-in on-air light in the chassis**.

<Danger>
  The **Tally Light** output automatically turns off if the load exceeds **150 mA**.
</Danger>

## GPIO

The 3.5mm jacks labeled \*\*GPIO\*\* are general-purpose outputs (**GPO**, _General Purpose Output_) that allow controlling external devices by enabling specific console channels.

- On **DX816**: they generate signals for microphone channels **MIC-1, MIC-2, MIC-3** and line channels **USB-1** and **USB-2**.
- On **DX822**: in addition to the above, there are also GPO outputs for **MIC-4, MIC-5 and MIC-6**.

<Note>
  The control outputs are **open-collector** type, functioning as a switch:

  - Inactive state → high impedance.
  - When enabling a channel → the output connects to ground.
</Note>

**Electrical specifications:**

- Maximum: **+24 V - 0.1 A**.

**Connectors:**

- **3.5mm TRS jack** type, located on the rear panel.
- On DX816: "MIC 1&2", "MIC-3" and "LIN USB 6&7".
- On DX822: additional "MIC 4&5" and "MIC-6" connectors are included.

---

**GPO MIC 1&2** _(same for MIC 4&5 on DX822)_

| Connector | Signal |
| -------- | ----- |
| Sleeve   | GND   |
| Ring     | MIC-1 |
| Tip      | MIC-2 |

**GPO MIC-3** _(same as MIC-6 on DX822)_

| Connector | Signal        |
| -------- | ------------ |
| Sleeve   | GND          |
| Ring     | MIC-3        |
| Tip      | Not connected |

**GPO USB**

| Connector | Signal |
| -------- | ----- |
| Sleeve   | GND   |
| Ring     | USB-1 |
| Tip      | USB-2 |
