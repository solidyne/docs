  <Accordion title="Audio">
    | Parameter              | Specification                                                                               |
    | --------------------- | ------------------------------------------------------------------------------------------- |
    | Frequency response    | 20–20,000 Hz ±0.5 dB (LIN -> PGM)                                                           |
    | MIC equalizer        | 4 bands: 80 Hz (shelving), 160 Hz (bell), 6.3 kHz (bell), 8 kHz (shelving), ±15 dB          |
    | MIC compressor       | Threshold 0VU, attack < 10 ms, ratio 10:1, max compression 20 dB, variation < 2 dB at +15 dB |
    | MIC noise            | EIN = –120 dBu / 150 Ω                                                                       |
    | Line noise           | S/N > 75 dBA                                                                                 |
    | Dynamic range        | >90 dBA (CD quality)                                                                         |
    | PGM–REC crosstalk    | >70 dBA @ 1 kHz                                                                             |
    | Distortion           | <0.03% THD (30–15,000 Hz)                                                                   |
    | Phase                | <3° L/R @ 50–10,000 Hz                                                                      |
    | Stereo correlation   | Error <0.2 dB L/R in 0–40 dB fader range                                                    |
    | Headroom             | 22 dB @ LIN -> PGM Ref +4 dBu / 10 kΩ                                                       |
  </Accordion>
  <Accordion title="VU-meters">

    - Quasi-peak measurement.
    - Compression indication: 0–15 dB.
  </Accordion>
