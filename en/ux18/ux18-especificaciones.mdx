  <Accordion title="General features">
    | Item                | Specification                                                      |
    | --------------------- | --------------------------------------------------------------------- |
    | Channels             | 18 channels                                                          |
    | Physical faders      | 10 on panel. 4 can work as groups (submixes)                         |
    | Virtual faders       | 8 Operable from touchscreen, assignable to groups                    |
    | Remote control       | Internal web interface (HTML5) via LAN                               |
    | Routing              | Inputs assignable to all 18 channels                                 |
    | Scenes               | 12 user scenes                                                       |
    | Touchscreen          | 7" — 1024 × 600 px, multi-touch                                     |
    | Channel displays     | OLED on channels 1 to 6 (name, input level, attenuation, status)    |
  </Accordion>
  <Accordion title="Inputs">
    #### Analog Inputs

    | Type          | Quantity | Details                                                                         |
    | ------------- | -------: | ------------------------------------------------------------------------------- |
    | Microphone    | 5        | +48 V phantom power switchable per channel                                      |
    | Stereo line   | 3        | 2 balanced + 1 unbalanced                                                       |
    | UNIDEX        | 4        | Configurable dual-mono / stereo. Supports UDX adapters (mic, Telco, USB, etc.) |

    #### Digital Inputs

    | Type                   | Quantity    | Details                                               |
    | ---------------------- | ----------: | ----------------------------------------------------- |
    | AES3                   | 2 (stereo)  | AES3 inputs support 44.1 and 48 kHz                  |
    | USB                    | 4 (stereo)  | 4 playback devices from a PC                         |
    | Bluetooth              | 1           | Mobile device pairing (Hands Free + A2DP music input) |
    | Dante/AES67 (optional) | 16          | Only in **UX18/A67** option                          |
  </Accordion>
  <Accordion title="Outputs">
    #### Analog Outputs

    | Type           | Quantity    | Details                             |
    | -------------- | ----------: | ----------------------------------- |
    | PGM            | 2 (stereo)  | 1 balanced + 1 unbalanced          |
    | REC            | 1           | Balanced                            |
    | Assignable     | 4 (stereo)  | Unbalanced                         |
    | Control Monitor | 2 (stereo)  | Speakers + headphones              |
    | Studio Monitor  | 2 (stereo)  | Speakers + headphones + talkback   |

    #### Digital Outputs

    | Type                   | Quantity    | Details                                   |
    | ---------------------- | ----------: | ----------------------------------------- |
    | AES3                   | 2 (stereo)  | AES3 outputs operate at 48 kHz            |
    | USB                    | 4 (stereo)  | 4 recording devices on a PC               |
    | Streaming              | 2           | RTP + Icecast/Shoutcast (independent)     |
    | Dante/AES67 (optional) | 16          | Only in **UX18/A67** option              |
  </Accordion>
  <Accordion title="GPIO, intercom and auxiliaries">
    | Item                              | Specification                                                |
    | --------------------------------- | ------------------------------------------------------------ |
    | "On Air" light                    | 12 VDC / 250 mA                                              |
    | GPI                               | 2 inputs (active high 3.3 ~ 15 V)                            |
    | GPO                               | 7 outputs (open collector; +24 V @ 250 mA)                   |
    | Air monitor (external tuner)      | 1 unbalanced stereo input (also assignable to channels)      |
    | Reverse talkback from Studio      | 1 input (Studio Box connector)                               |
    | Talkback                          | Built-in talkback microphone                                 |
    | TELCO intercom                    | PTT (talk/listen) on TELCO channels with CUE button          |
  </Accordion>
  <Accordion title="Audio architecture">
    | Item                 | Specification                                                                |
    | -------------------- | --------------------------------------------------------------------------- |
    | Buses                | 4 stereo (PGM, REC, AUX, VBUS)                                             |
    | PFL Bus              | 1 (CUE)                                                                     |
    | Resolution / SR      | 24 bits @ 48 kHz                                                            |
    | Processing           | DSP on all inputs (fixed per input)                                         |
    | Processes            | 3-band parametric EQ, gate, compressor/limiter, de-esser                    |
    | Mix-minus / talkback | Automatic feeds for TELCO                                                   |
    | Latency             | 1.2 ms (line→analog out) / 1.4 ms (MIC→output with EQ+Dynamics+De-Esser)    |
  </Accordion>
