  <Accordion title="Inputs">
    #### Analog Inputs

    | Type | Quantity | Detail |
    | --- | --: | --- |
    | Microphone | 5 | Switchable \+48 V phantom power per channel |
    | Stereo line | 3 | 2 balanced \+ 1 unbalanced |
    | UNIDEX | 8 mono / 4 stereo | Balanced line-level inputs with extended capabilities for connecting Solidyne UDX adapters. Support up to 8 additional sources (microphones, TELCO lines). |

    #### Digital Inputs

    | Type | Quantity | Detail |
    | --- | --: | --- |
    | AES3 | 2 (stereo) | Support sample rates of 44.1 - 48 - 96 kHz |
    | USB Audio | 4 stereo / 8 mono | 4 playback devices from a PC (WDM).<br />ASIO compatible → supports 8-IN x 8-OUT mode |
    | Bluetooth | 1 | Mobile device pairing (Hands Free \+ A2DP music input) |
    | Dante/AES67 (optional) | 16 (mono) | Available in **UX18/A67** option only |
  </Accordion>

  <Accordion title="Audio Architecture and Processing">
    | Item | Specification |
    | --- | --- |
    | Buses | 4 stereo (PGM, REC, AUX, VBUS) |
    | PFL bus | 1 (CUE) |
    | Resolution / SR | 24 bits @ 48 kHz |
    | Processing | DSP on all inputs (fixed per input) |
    | Processes | 3-band parametric EQ, gate, compressor/limiter, de-esser |
    | Mix-minus / talkback | Automatic sends for TELCO |
    | Latency | 1.2 ms (line→analog output) / 1.4 ms (MIC→output with EQ\+Dynamics\+De-Esser) |
  </Accordion>
