  <Accordion title="Arquitectura de audio">
    | Ítem                 | Especificación                                                                 |
    | -------------------- | ------------------------------------------------------------------------------ |
    | Buses                | 4 estéreo (PGM, REC, AUX, VBUS)                                                |
    | Bus PFL              | 1 (CUE)                                                                        |
    | Resolución / SR      | 24 bits @ 48 kHz                                                               |
    | Procesado            | DSP en todas las entradas (fijo por entrada)                                   |
    | Procesos             | EQ paramétrico 3 bandas, compuerta, compresor/limitador, de-esser              |
    | Mix-minus / talkback | Envíos automáticos para TELCO                                                  |
    | Latencia             | 1.2 ms (línea→salida analógica) / 1.4 ms (MIC→salida con EQ+Dinámica+De-Esser) |
  </Accordion>
