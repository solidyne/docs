## Firmware Update

The console software can be updated from the WEB interface using a computer with Internet access.

Updates may include:

- Mixing engine firmware (hardware).
- Graphical interface.
- Streaming tools.

## Update Procedure

### 1. Network Connection

1. Connect the console's **"Streaming Out"** and **"Remote Control"** Ethernet ports to the local network.
2. Verify that the network has Internet access.
3. On the main screen (VU meters), verify the IP address assigned to the streaming stage.

<discovery />

### 2. Accessing the Update Screen

From a computer connected to the same network:

1. Open a web browser.
2. Enter the console's IP address.
3. The update screen will open.
4. In the left menu, select UPDATE.
5. The update screen will load in the right panel.

<img
  src="/images/ux24/ux24-update-firmware.png"
  alt="UNIDEX Firmware Update"
  title="UNIDEX Firmware Update"
  style={{ width:"61%" }}
/>

4. Click **CHECK FOR UPDATES**.
3. If updates are available, a list with details will be displayed.
4. The **INSTALL** option starts the update process.

<Warning>
  The system **NEVER INSTALLS UPDATES AUTOMATICALLY**. Installing an update requires operator intervention.
</Warning>

<Info>
  Depending on the type of update, the unit may restart, **INTERRUPTING** the on-air signal for a few seconds.
</Info>
