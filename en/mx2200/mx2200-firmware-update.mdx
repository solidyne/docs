The system update is performed from the console's web interface:

<Steps>
  <Step title="Access the update system">
    Enter the console's IP address in the browser followed by the subpath **/update**.\
    Example: `http://192.168.0.121/update`
  </Step>
  <Step title="Check available packages">
    The screen displays the installed software packages and, if available, the latest versions ready for update.
  </Step>
  <Step title="Download update">
    Click **Download** to get the latest version.\
    The package is stored in the **console's internal memory**.
  </Step>
  <Step title="Install update">
    Once the download is complete, the **Install** option becomes available.\
    Click to start the installation process.
  </Step>
</Steps>

<Info>
  - The console allows updates even over **Wi-Fi**.
  - Since the package is stored locally before installation, the console can complete the installation autonomously.
  - The **Hardware Info** section displays **hardware** information such as battery charge cycles.
</Info>
